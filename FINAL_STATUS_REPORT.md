## 🎯 Осталось 3 шага для 100% готовности

### Шаг 1: Деплой Backend на Render.com (15 минут)

**Инструкция:** `RENDER_DEPLOY.md`

**Кратко:**
```bash
# 1. Запушить код на GitHub
git add .
git commit -m "Production ready with Supabase"
git push origin main

# 2. Создать сервис на render.com
# - dashboard.render.com → New Web Service
# - Подключить GitHub репозиторий
# - Конфигурация будет автоматически загружена из render.yaml

# 3. Дождаться деплоя (3-5 минут)
# Получить URL: https://lumen-bank-api.onrender.com

# 4. Проверить
curl https://lumen-bank-api.onrender.com/health
```

### Шаг 2: Обновить конфигурацию и пересобрать APK (10 минут)

```bash
# 1. Обновить root .env
echo "VITE_BACKEND_URL=https://lumen-bank-api.onrender.com" > .env

# 2. Пересобрать frontend
npm run build

# 3. Синхронизировать Capacitor
npm run cap:sync

# 4. Собрать новый APK
cd android
./gradlew assembleDebug

# Новый APK будет здесь:
# android/app/build/outputs/apk/debug/app-debug.apk
```

### Шаг 3: Загрузка на Loadly.io (5 минут)

1. Открыть [loadly.io](https://loadly.io)
2. Загрузить новый APK
3. Получить QR код и ссылку
4. Разослать студентам
