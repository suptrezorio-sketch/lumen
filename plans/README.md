# LUMEN AI Instructions Pack

## Назначение

Этот пакет `.md` файлов — инструкция для ИИ/разработчика, который должен перестроить LUMEN:

- клиентское приложение банка
- кастомную CRM / Bank Control Center
- PocketBase backend
- структуру данных
- KYC/AML
- операции и ledger
- smart contract / settlement order
- Wallet Risk Training
- Framework7 дизайн-систему
- единый input/keyboard system
- PDF receipts, push, in-app simulated bank call

## Главный принцип

LUMEN должен ощущаться как полноценное банковское приложение, а не как игрушечный прототип.

При этом система работает в контролируемой учебной среде:

- финансовые сущности существуют во внутреннем ledger
- карты, кошельки, счета и контракты не должны быть просто текстом на экране
- все операции имеют статусы
- CRM управляет каждым клиентом, состоянием, сценарием, картой, кошельком и операцией

## Как использовать

Дай ИИ весь zip или вставь файлы по порядку.

Главный файл для старта:

```text
12_MASTER_AI_REFACTOR_PROMPT.md
```

Затем ИИ должен читать остальные файлы по номерам.

## Критически важно

Не просить ИИ “сделай красиво”. Просить ИИ следовать этим файлам.
